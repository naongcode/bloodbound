# 던전 보상 상세 기획서

---

## 1. 보상 구조 개요

던전 보상은 3단계로 구성된다.

```
[층 클리어 보상]  →  [보스 처치 보상]  →  [던전 완료 보상]
  각 층 클리어 시        각 층 보스 처치 시      전 층 클리어 후
  자동 지급              드롭 발생               완료 상자 지급
```

- 모든 보상은 **파티원 각자에게 개별 지급** (기본 분배 방식)
- 단, 드롭 장비는 분배 방식 설정에 따라 상이

|------|------------|--------------|
| 서광의 동굴 | 일일 초기화 | 1일 1회 추가 보상 |
| 안개 숲 폐허 | 일일 초기화 | 1일 1회 추가 보상 |
| 붉은 사막 지하 미궁 | 주간 초기화 | 주 3회 추가 보상 |

## 4. 드롭 장비 직업 연동

보스 드롭 장비는 **파티 내 직업 구성에 따라 드롭 풀이 조정**됨.

- 파티에 전사 계열 있으면: 전사용 무기/갑옷 드롭률 +30%
- 파티에 마법사 계열 있으면: 마법사용 스태프 드롭률 +30%
- 파티에 사제 계열 있으면: 사제용 장신구 드롭률 +30%

> 파티 구성에 없는 직업의 장비도 드롭 가능 (거래 가능).

## 5. 특별 보상 — 칭호

던전 특정 조건 달성 시 **고유 칭호** 획득. 닉네임 옆에 표시 가능.

| 칭호 | 획득 조건 |
|------|-----------|
| "서광의 탐험가" | 서광의 동굴 S등급 클리어 |
| "숲의 정복자" | 안개 숲 폐허 10회 클리어 |
| "사막의 생존자" | 붉은 사막 지하 미궁 솔로 클리어 |
| "탑의 등반자" | 심연의 탑 50회 클리어 |
| "균열의 봉인자" | 균열의 핵 첫 클리어 (서버 전체 공지) |
| "무결한 등반" | 심연의 탑 전원 생존 S등급 클리어 |
| "피를 이긴 자" | 흡혈 저항 200 이상으로 나이로스 처치 |
