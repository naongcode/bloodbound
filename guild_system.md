# 길드 시스템 기획서

---

## 1. 개요

길드는 단순한 소속 집단이 아닌 **함께 키워나가는 성장형 조직**이다.
길드 레벨이 오를수록 구성원 전원이 실질적인 혜택을 받으며, 고레벨 길드일수록 던전 공략에 유리해진다.

---

## 2. 길드 창설 및 기본 구조

| 항목 | 내용 |
|------|------|
| 창설 조건 | 캐릭터 레벨 20 이상, 창설 비용 5,000 골드 |
| 최대 인원 | 길드 레벨에 따라 확장 (기본 20명 → 최대 100명) |
| 길드 레벨 범위 | 1 ~ 30 |
| 직책 구조 | 길드마스터 → 부길드마스터 → 장교 → 조합원 → 신입 |

### 2.1 직책별 권한

| 직책 | 권한 |
|------|------|
| 길드마스터 | 모든 권한. 길드 해산, 마스터 위임. |
| 부길드마스터 | 가입 승인/거부, 추방, 공지 작성. |
| 장교 | 길드 창고 아이템 입출고, 이벤트 소집. |
| 조합원 | 길드 창고 열람, 길드 퀘스트 참여. |
| 신입 | 길드 채팅, 길드 버프 수령. |

## 3. 길드 성장 방법

### 3.1 길드 경험치 획득 수단

길드 경험치(Guild EXP)는 구성원들의 활동으로 쌓인다.

| 활동 | 획득 길드 EXP |
|------|--------------|
| 길드 퀘스트 완료 | 500 ~ 3,000 |
| 던전 클리어 (파티 중 길드원 3인 이상) | 1,000 ~ 5,000 |
| 주간 길드 레이드 참여 | 5,000 ~ 20,000 |
| 길드원 첫 레벨업 (일 1회) | 100 |
| 길드 전쟁 승리 | 10,000 |
| 길드 건물 업그레이드 완료 | 2,000 ~ 10,000 |

### 3.2 길드 기여도 (Contribution)

- 각 구성원은 위 활동 시 **개인 기여도**도 함께 쌓는다.
- 기여도는 길드 상점에서 아이템 구매 통화로 사용된다.
- 기여도 랭킹은 주간/월간/전체 기준으로 표시된다.
- 낮은 기여도 구성원은 장교 이상이 경고 및 추방 가능.

### 3.3 길드 건물 (Guild Hall)

길드는 전용 **길드 홀(Guild Hall)**을 보유하며, 골드와 심연석을 투자해 건물을 업그레이드한다.

| 건물 | 기본 기능 | 업그레이드 효과 |
|------|-----------|----------------|
| 길드 홀 본관 | 길드 최대 인원 결정 | Lv당 +5명 증가 |
| 훈련소 | 길드원 EXP 획득량 보너스 | 최대 +20% EXP |
| 대장간 | 길드 전용 장비 제작 | 제작 가능 레시피 확장 |
| 도서관 | 스킬 강화 연구 | 특정 스킬 쿨타임/위력 개선 |
| 창고 | 공용 아이템 보관 | 슬롯 수 확장 |
| 치유소 | 던전 입장 전 버프 제공 | 버프 강도 및 지속시간 증가 |
| 망루 | 길드 전쟁 방어 보너스 | 방어 포인트 증가 |

## 4. 길드 레벨별 혜택

### 4.1 패시브 혜택 (자동 적용)

| 길드 레벨 | 혜택 |
|-----------|------|
| Lv 1 | 길드 채팅, 길드 창고(기본), 길드 퀘스트 해금 |
| Lv 3 | 골드 획득량 +3% |
| Lv 5 | 최대 인원 +10명, 아이템 드롭률 +2% |
| Lv 7 | 경험치 획득량 +5% |
| Lv 10 | 길드 레이드 던전 해금, 이동속도 +3% |
| Lv 12 | 골드 획득량 +6%, 드롭률 +4% |
| Lv 15 | 경험치 획득량 +10%, 길드 대장간 2단계 해금 |
| Lv 18 | 피의 정수 드롭률 +5% (흡혈 몬스터 전용) |
| Lv 20 | 파티 던전 클리어 보너스 보상 +10%, 길드 전쟁 해금 |
| Lv 25 | 전설 장비 제작 레시피 해금, 흡혈 저항 +5 (전 구성원) |
| Lv 30 | 최종 레이드 전용 버프 해금, 골드/드롭/EXP 전 보너스 최대치 |

### 4.2 길드 버프 (길드마스터/장교가 수동 발동)

- 1일 1회 발동 가능, 비용은 길드 자금(Guild Fund)에서 차감
- 버프는 길드원 전원에게 적용, 지속시간 2시간

| 버프 이름 | 효과 | 비용 |
|-----------|------|------|
| 전투의 기운 | 공격력 +10% | 3,000 골드 |
| 방어의 의지 | 방어력 +10%, 흡혈 저항 +5 | 3,000 골드 |
| 탐험가의 본능 | 드롭률 +5%, 이동속도 +5% | 4,000 골드 |
| 심연의 각성 | 스킬 쿨타임 -10% | 5,000 골드 |
| 혈맹의 서약 | 위 효과 전부 절반씩 + 경험치 +10% | 15,000 골드 |

## 5. 길드 콘텐츠

### 5.1 길드 퀘스트

- 매일 초기화되는 길드 전용 퀘스트 목록 (3~5개)
- 개인이 수행 후 길드 EXP + 개인 기여도 + 골드 보상
- 예시: "흡혈 늑대 50마리 처치", "던전 2회 클리어", "보스 1회 처치"

### 5.2 길드 레이드 (주간)

- 길드 레벨 10 해금
- 최대 10~20명 참여 가능한 대규모 레이드 던전
- 주 1회 도전 가능, 클리어 시 길드 전원에게 특수 보상 우편 발송
- 레이드 보스는 **흡혈 대군주** 계열의 강력한 보스

### 5.3 길드 전쟁 (Guild War)

- 길드 레벨 20 해금
- 다른 길드에 선전포고 → 72시간 내 점수 경쟁
- 점수 획득 방법: 상대 길드원 처치, 거점 점령, 미션 완료
- 승리 시: 길드 EXP 대량 획득 + 전쟁 트로피 (길드 홀 장식)
- 패배 시: 길드 EXP 소량 획득 (참여 보상)

### 5.4 길드 상점

- 기여도로만 구매 가능한 전용 아이템 보유
- 상점 품목 예시:

| 아이템 | 기여도 비용 |
|--------|-------------|
| 강화석 (고급) | 500 |
| 흡혈 저항 반지 (희귀) | 2,000 |
| 길드 전용 코스튬 (직업별) | 10,000 |
| 전설 장비 제작 재료 | 5,000 |
| 길드 문장 문양 (칭호) | 3,000 |

---

## 6. 길드 자금 (Guild Fund)

- 구성원이 골드를 직접 기부해 적립
- 길드 버프 발동, 건물 업그레이드, 레이드 소모품 구매에 사용
- 기부 금액도 기여도로 일부 환산 (1,000 골드 기부 = 기여도 50)

---

## 7. 길드 성장 흐름 요약

```
[창설] → 퀘스트·던전 활동으로 Guild EXP 축적
       → 레벨업 시 패시브 혜택 자동 해금
       → 골드/심연석으로 길드 건물 업그레이드
       → 고레벨 콘텐츠(레이드, 전쟁) 참여
       → 길드 상점에서 고급 아이템 획득
       → 구성원 전체 전투력 상승 → 더 깊은 던전 공략 가능
```
