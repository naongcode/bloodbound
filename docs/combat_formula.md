## 1. 기본 피해 계산

### 1.1 물리 피해

```
물리 피해 = (공격자 STR × 2.5 + 무기 공격력) × 스킬 배율 × 랜덤(0.9 ~ 1.1)
           - 방어력 감소량
```

- **방어력 감소량** = 방어력 / (방어력 + 대상 레벨 × 10) × 물리 피해
- 최소 보장 피해: 계산 후 피해가 1 미만이면 1로 처리

### 1.2 마법 피해

```
마법 피해 = (공격자 INT × 3.0 + 스킬 기본 위력) × 스킬 배율 × 랜덤(0.9 ~ 1.1)
           × (1 - 마법 저항률)
```

- **마법 저항률** = 마법 저항 / (마법 저항 + 200) (최대 75%)

### 1.3 스킬 배율 기준값

| 등급 | 배율 범위 | 예시 |
|------|-----------|------|
| 약 | 0.5 ~ 0.8 | 빠른 연타, 보조기 |
| 보통 | 1.0 ~ 1.5 | 기본 스킬 |
| 강 | 2.0 ~ 3.0 | 차지 스킬, 필살기 |
| 범위 | 0.6 ~ 1.0 | 광역 공격 (다중 히트) |

### 2.4 HP / MP 계산

```
최대 HP = (VIT × 20) + (레벨 × 50) + 기본 HP
최대 MP = (WIS × 15) + (레벨 × 20) + 기본 MP
```

|-----------|---------|---------|

| 마법사 계열 | 200 | 400 |
| 사제 계열 | 250 | 350 |
| 연금술사 계열 | 300 | 250 |

## 3. 회피 / 명중 계산

### 3.1 명중률

```
명중률 = 75% + (공격자 AGI - 대상 AGI) × 0.5%
```
- 최소 명중률: 20%
- 최대 명중률: 99%

### 3.2 회피율

```
회피율 = AGI × 0.1% + 장비 회피 보너스
```
- 최대 회피율: 50% (상한선)

### 3.3 치명타

```
치명타 확률 = AGI × 0.05% + 장비 치명타 보너스
치명타 배율 = 150% + 장비 치명타 피해 보너스
```
- 기본 치명타 배율: 150%
- 최대 치명타 확률: 60%

## 4. 흡혈 수치 계산

### 4.1 몬스터 흡혈 회복량

```
흡혈 회복 HP = 가한 피해 × 흡혈률 × (1 - 대상 RES / (RES + 200))
```

| 몬스터 등급 | 기본 흡혈률 |
|-------------|-------------|
| 일반 | 20% |
| 정예 | 35% |
| 챔피언 | 50% |
| 보스 | 60% |
| 레이드 보스 | 70% |

### 4.2 흡혈 저항 (RES) 효과

| RES 값 | 흡혈 감소율 | 실효 흡혈률 (일반 몬스터 기준) |
|--------|------------|-------------------------------|
| 0 | 0% | 20% |
| 50 | 20% | 16% |
| 100 | 33% | 13.4% |
| 150 | 43% | 11.4% |
| 200 | 50% | 10% |
| 300 | 60% | 8% |
| 500 | 71% | 5.8% (상한 근접) |

- RES 무한대 이론값: 최대 약 80% 감소 (실제 상한: 80%)

### 4.3 광역 흡혈 오라 (지속 피해)

```
오라 초당 HP 흡수 = 오라 기본값 × (1 - RES / (RES + 300))
```

- 오라 기본값은 몬스터/던전 레벨에 따라 결정
- 오라 범위 내 플레이어 수가 많을수록 몬스터 회복량 증가 (파티 압박 설계)

## 5. 힐 / 회복 계산

### 5.1 스킬 힐

```
힐량 = (시전자 WIS × 3.5 + 스킬 기본 힐량) × 스킬 배율
```

### 5.2 HP 자연 회복

```
초당 HP 회복 = VIT × 0.1 (전투 외)
전투 중 자연 회복 = VIT × 0.02
```

### 5.3 MP 자연 회복

```
초당 MP 회복 = WIS × 0.15 (전투 외)
전투 중 MP 회복 = WIS × 0.05
```

## 6. 상태이상 수치

| 상태이상 | 효과 | 지속시간 기준 | 저항 방법 |
|----------|------|--------------|-----------|
| 중독 | 초당 INT × 0.5 피해 | 5~15초 | 해독 포션, 사제 정화 |
| 화상 | 초당 INT × 0.8 피해 (불 속성) | 3~8초 | 물 속성 공격, 포션 |
| 빙결 | 이동속도 -70%, 공격속도 -50% | 2~5초 | 불 속성 피격 시 해제 |
| 속박 | 이동 불가 | 1~4초 | 탈출 스킬, 파티원 구조 |
| 경직 | 모든 행동 불가 | 0.5~2초 | 없음 (짧은 시간) |
| 흡혈 저주 | 5초마다 최대 HP × 2% 감소 | 20~30초 | 사제 정화, 전용 포션 |
| 격노 (몬스터) | 공격력+50%, 방어력-20% | 파훼 실패 시 영구 | 해제 불가 (전투 압박) |
| 약화 | 공격력-30%, 방어력-30% | 5~10초 | 자연 해제 |

## 7. 경험치 공식

### 7.1 몬스터 처치 경험치

```
기본 EXP = 몬스터 기준 EXP × 레벨 보정
레벨 보정 = max(0.1, 1 - (플레이어 레벨 - 몬스터 레벨) × 0.05)
```

- 플레이어가 몬스터보다 10레벨 이상 높으면 EXP 급감
- 몬스터가 플레이어보다 높으면 보정 없음 (100%)

### 7.2 파티 경험치 분배

```
파티 EXP 배율 = 1.0 + 파티원 수 × 0.1
개인 획득 EXP = (기본 EXP × 파티 EXP 배율) / 파티원 수
```

| 파티 인원 | EXP 배율 | 개인 획득 (솔로 대비) |
|-----------|----------|----------------------|
| 1인 (솔로) | ×1.0 | 100% |
| 2인 | ×1.2 | 60% |
| 3인 | ×1.3 | 43% |
| 4인 | ×1.4 | 35% |
| 5인 | ×1.5 | 30% |

> 솔로 대비 파티는 EXP 효율이 낮지만, 파티 전용 던전 보상으로 보정된다.

### 7.3 레벨별 누적 경험치 요구량

```
필요 EXP(Lv N) = 100 × N^1.8
```

| 레벨 | 해당 레벨 필요 EXP | 누적 EXP |
|------|-------------------|---------|
| 2 | 180 | 180 |
| 10 | 6,310 | 약 30,000 |
| 20 | 21,120 | 약 150,000 |
| 30 | 42,400 | 약 430,000 |
| 50 | 100,000 | 약 1,500,000 |
| 80 | 220,000 | 약 5,500,000 |
| 100 | 316,000 | 약 9,500,000 |

## 8. 장비 스탯 보정 범위

### 8.1 주 스탯 (장비 레벨 기준)

```
장비 주 스탯 = 장비 레벨 × 등급 배율 × (0.9 ~ 1.1 랜덤)
```

| 등급 | 배율 |
|------|------|
| 일반 | ×1.0 |
| 고급 | ×1.3 |
| 희귀 | ×1.7 |
| 영웅 | ×2.2 |
| 전설 | ×3.0 |
| 심연 | ×3.5 + 특수 효과 |

### 8.2 강화 보너스

```
강화 스탯 보너스 = 기본 스탯 × (강화 단계 × 0.06)
```

| 강화 단계 | 스탯 보너스 | 실패 확률 | 실패 시 효과 |
|-----------|-------------|-----------|-------------|
| +1 ~ +5 | +6% ~ +30% | 0% | 없음 |
| +6 ~ +9 | +36% ~ +54% | 10% ~ 25% | 강화 단계 -1 |
| +10 ~ +12 | +60% ~ +72% | 40% ~ 55% | 등급 하락 가능 |
| +13 ~ +15 | +78% ~ +90% | 70% ~ 85% | 등급 하락 or 파괴 |
