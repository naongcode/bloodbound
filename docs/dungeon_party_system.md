# 던전 매칭 / 파티 시스템 기획서

## 1. 파티 시스템

### 1.1 파티 구성

| 항목 | 내용 |
|------|------|
| 최소 인원 | 1인 (솔로 가능) |
| 최대 인원 | 5인 |
| 파티 리더 | 파티 생성자. 던전 입장 결정권. |
| 파티 리더 위임 | 리더가 파티원에게 언제든 위임 가능 |

### 1.2 파티 구성 방법

**방법 A — 직접 초대**
1. 같은 필드에 있는 플레이어에게 우클릭 → "파티 초대"
2. 캐릭터명 직접 입력 후 초대 (다른 필드에 있어도 가능)
3. 초대받은 플레이어가 수락/거절

**방법 B — 파티 찾기 (공개 모집)**
1. 파티 게시판(UI)에서 파티 모집글 등록
   - 던전 이름, 권장 직업, 권장 레벨, 메모 입력
2. 다른 플레이어가 게시글 확인 후 참가 신청
3. 파티 리더가 승인

**방법 C — 랜덤 매칭 (자동)**
- 던전 선택 후 "자동 매칭" 버튼 클릭
- 시스템이 비슷한 레벨의 플레이어 자동 연결
- 매칭 중에는 현재 필드에서 계속 플레이 가능
- 매칭 완료 시 알림 → 30초 내 수락하면 입장

### 1.3 파티 UI

```
┌─────────────────────────────────────┐
│  파티 (3/5)            [파티 해산]  │
├─────┬───────────┬────────┬──────────┤
│ 아이콘 │  닉네임    │ 직업   │ HP/MP   │
├─────┼───────────┼────────┼──────────┤
│  ●  │ Naong (리더)│ 기사   │ ████░░  │
│  ●  │ Mage01    │ 화염술사│ ████████│
│  ●  │ Priest99  │ 치유사  │ ██████░ │
│  ○  │ (빈 슬롯)  │        │         │
│  ○  │ (빈 슬롯)  │        │         │
└─────┴───────────┴────────┴──────────┘
```

- 파티원 HP/MP 실시간 표시 (사제 힐 타겟 확인용)
- 사망한 파티원은 빨간색으로 표시
- 파티원 클릭 시 해당 플레이어에게 스킬/힐 사용 가능

## 2. 던전 시스템

### 2.1 던전 유형

| 유형 | 인원 | 특징 |
|------|------|------|
| **솔로 던전** | 1인 | 개인 파밍, 일일 퀘스트 연계 |
| **파티 던전** | 2~5인 | 메인 던전 콘텐츠. 인스턴스 분리. |
| **길드 레이드** | 10~20인 | 주간 1회. 길드 전용. |
| **공개 던전** | 제한 없음 | 인스턴스 없음. 여러 파티가 같은 공간. |

### 2.2 던전 목록

| 던전 이름 | 유형 | 권장 레벨 | 인원 | 층 수 |
|-----------|------|-----------|------|-------|
| 서광의 동굴 | 솔로/파티 | 10~20 | 1~3인 | 3층 |
| 안개 숲 폐허 | 파티 | 25~40 | 2~5인 | 5층 |
| 붉은 사막 지하 미궁 | 파티 | 40~55 | 3~5인 | 7층 |
| 심연의 탑 | 파티 | 55~80 | 4~5인 | 10층 |
| 혈족 요새 | 파티 | 65~80 | 4~5인 | 8층 |
| 균열의 핵 | 파티 | 80~100 | 5인 | 5층 |
| 심연 대군주의 성 | 길드 레이드 | 85~100 | 10~20인 | 3층 |

### 2.3 인스턴스 구조

- 파티가 던전에 입장하면 **전용 인스턴스(Instance)** 생성
- 다른 파티와 몬스터/보스를 공유하지 않음
- 인스턴스는 파티 전원 사망 또는 던전 클리어 시 종료
- 파티 해산 시 인스턴스 강제 종료 (5분 유예 시간 후)

## 3. 던전 입장 흐름

```
[던전 입장 포탈 상호작용]
          │
    파티 리더 여부 확인
     ┌────┴────┐
   리더       비리더
     │           │
 던전 선택    "리더에게 입장 요청" 팝업
     │
 파티원 준비 확인 (Ready Check)
     │
 전원 Ready → 입장 카운트다운 10초
     │
 인스턴스 생성 → 파티 전원 던전 이동
```

### 3.1 준비 확인 (Ready Check)

- 파티 리더가 "입장 준비" 클릭 시 모든 파티원에게 팝업
- 30초 내 전원 수락 시 입장
- 미응답 또는 거절 시 해당 파티원 자동 제외 (리더 선택)

### 3.2 재입장

- 던전 진행 중 파티원 접속 끊김 → 10분 내 재접속 시 복귀 가능
- 10분 초과 시 해당 파티원 없이 진행
- 보스 처치 후 입장한 신규 파티원은 해당 보스 보상 미지급 (입장 시점 기준)

## 4. 던전 진행 구조

### 4.1 층 구조

```
[입장]
  │
 일반 몬스터 구역 (클리어해야 다음 진행)
  │
 선택 전투 구역 (추가 보상 but 위험)
  │
 중간 보스 방 (층 클리어 보스)
  │
 보상 상자 + 다음 층 포탈
  │
[최종 층]
  │
 레이드 보스 방
  │
 클리어 보상 + 던전 완료
```

### 4.2 층 클리어 조건

- **기본**: 해당 층의 모든 필수 몬스터 처치
- **선택 구역**: 진입 여부를 파티 투표로 결정 (리더 결정권)
- 시간 내 클리어 시 추가 보상 (빠른 클리어 등급)

### 4.3 클리어 등급

| 등급 | 조건 | 추가 보상 |
|------|------|-----------|
| S | 권장 시간 70% 이내 클리어, 전원 생존 | 심연석 추가 + 희귀 이상 장비 드롭률 +30% |
| A | 권장 시간 100% 이내 클리어 | 심연석 추가 |
| B | 클리어 (시간 초과 가능) | 기본 보상 |
| F | 전원 사망 (클리어 실패) | 보상 없음, 경험치 패널티 |

## 5. 던전 내 규칙

### 5.1 아이템 드롭 및 분배

| 방식 | 설명 |
|------|------|
| **자동 분배** (기본) | 보스 처치 시 파티원 전원에게 각자 드롭 아이템 지급 (복사 지급) |
| **주사위 분배** | 아이템 드롭 시 파티원 전원이 주사위 굴림, 최고값 플레이어 획득 |
| **리더 지정** | 리더가 직접 아이템 수령자 지정 |

- 파티 입장 전 리더가 분배 방식 선택
- 기본 추천: **자동 분배** (모두에게 동일 아이템 복사 지급)

### 5.2 부활

- 던전 내 사망 시 해당 위치에서 부활 불가
- 파티원의 **부활 스킬** 또는 **부활 아이템**으로만 소생 가능
- 소생 없을 시 입구에서 유령 상태로 대기 (이동만 가능, 전투 불가)
- 전원 사망 시 던전 실패

### 5.3 포션 제한

- 던전 내 포션 사용 간격: 동일 종류 5초 쿨타임
- 던전 레벨에 따른 포션 등급 제한 (고레벨 던전 = 고급 포션만 효과)

## 6. 파티 권장 구성 가이드

### 6.1 역할군

| 역할 | 담당 직업 | 역할 설명 |
|------|-----------|-----------|
| **탱커 (Tank)** | 기사, 전사, 신성기사 | 어그로 집중, 방어 전담 |
| **딜러 (DPS)** | 버서커, 궁수, 암살자, 레인저, 화염술사, 빙결술사, 폭탄사 | 몬스터 처치 주력 |
| **서포터 (Support)** | 치유사, 사제, 독술사 | 힐, 버프, 디버프, 해제 |

### 6.2 던전별 권장 파티 구성

| 던전 | 권장 구성 | 이유 |
|------|-----------|------|
| 서광의 동굴 | 자유 | 쉬운 난이도 |
| 안개 숲 폐허 | 탱1 딜3 서포1 | 광역 흡혈 오라 첫 등장 → 힐러 필요 |
| 붉은 사막 지하 미궁 | 탱1 딜2 서포2 | 마나 흡혈 몬스터 → 해제/보호 필요 |
| 심연의 탑 | 탱1 딜2 서포2 또는 탱1 딜3 서포1 | 완전 방어 보유 몬스터 → 협동 파훼 필수 |
| 균열의 핵 | 탱1 딜2 서포2 | 레이드 보스 흡혈 오라 + 즉사기 → 사제 필수 |

### 6.3 솔로 플레이 가능 기준

| 던전 | 솔로 가능 여부 | 비고 |
|------|---------------|------|
| 서광의 동굴 | 가능 | |
| 안개 숲 폐허 | 고레벨 가능 | 권장보다 15레벨 이상 높으면 가능 |
| 붉은 사막 지하 미궁 | 어렵지만 가능 | 연금술사 직업 유리 |
| 심연의 탑 | 불가 | 완전 방어 파훼 조건 충족 불가 |
| 균열의 핵 | 불가 | 즉사기 무효화 조건 충족 불가 |

## 7. 파티 채팅 및 소통

| 채팅 채널 | 범위 | 단축키 |
|-----------|------|--------|
| 일반 채팅 | 같은 맵 전체 | Enter |
| 파티 채팅 | 파티원만 | P + Enter |
| 길드 채팅 | 길드원 전체 | G + Enter |
| 귓속말 | 1:1 | /닉네임 메시지 |

### 빠른 소통 (핑 시스템)

- 미니맵 클릭 → 위치 핑 (파티원에게 표시)
- 몬스터 클릭 → "이 몬스터 공격" 핑
- 자신 클릭 → "나 좀 도와줘" 핑 (사제 힐 요청)
- 단축 메시지 휠: "준비됨", "잠깐 기다려", "방어 파훼!", "힐 필요!" 등 8종
