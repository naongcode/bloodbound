# 게임 기획서 — 《흡혈의 땅 (Bloodbound Realm)》

---

| 항목 | 내용 |
|------|------|

## 2. 세계관

### 2.1 배경 설정

세계 이름: **아스란 대륙 (Continent of Aslan)**

수백 년 전, 아스란 대륙의 중심부에 "심연의 균열(Abyssal Rift)"이 열렸다. 균열에서 쏟아져 나온 어둠의 기운은 짐승과 마물들을 변이시켰고, 그 중 가장 강력한 존재들은 **흡혈의 저주(Blood Curse)**를 얻었다 — 살아있는 생명체의 피와 생명력을 흡수해 스스로를 강화하는 능력이다.

균열로부터 퍼진 저주는 느리지만 확실히 대륙 전역으로 번지고 있다. 인간 왕국들은 연합 탐험대를 구성해 균열의 근원을 찾아 봉인하고자 하지만, 수많은 탐험대가 실패를 거듭하고 있다.

플레이어는 이 세계의 **탐험가(Explorer)**로서, 평시에는 각지의 마을과 야외에서 의뢰를 수행하며 성장하고, 충분히 강해지면 다른 탐험가들과 함께 심연의 던전으로 깊이 들어가 균열의 원흉을 처치한다.

### 2.2 주요 세력

| 세력 | 설명 |
|------|------|
| 아스란 연합 왕국 | 인간 왕국 연합. 탐험가 길드를 후원한다. |
| 탐험가 길드 | 플레이어 소속 조직. 퀘스트·장비 거래 허브. |
| 심연의 군주단 | 흡혈 저주에 물든 괴물 지배 계급. 최종 보스 집단. |
| 혈족(Blood Kin) | 저주에 자발적으로 몸을 내준 배신자 인간 집단. 중간 보스 역할. |

### 2.3 지역 구성

| 지역 | 특징 |
|------|------|
| 서광 평원 | 튜토리얼 지역. 흡혈 슬라임과 약한 박쥐류 등장. |
| 안개 숲 | 중급 지역. 흡혈 늑대, 뱀파이어 나무 등장. |
| 붉은 사막 | 중상급 지역. 흡혈 벌레, 사막 뱀파이어 도마뱀 등장. |
| 심연의 탑 | 고급 던전 지역. 4~5인 파티 전용. 다층 구조. |
| 균열의 핵 | 최종 던전. 심연의 군주단 보스 등장. |

## 3. 캐릭터 시스템

### 3.1 직업 (Job)

초기 직업 선택 후 레벨 업 시 상위 직업으로 전직 가능.

| 계열 | 1차 직업 | 2차 직업 | 설명 |
|------|---------|---------|------|
| 전사 계열 | 전사 (Warrior) | 기사 / 버서커 | 근접 탱커. 높은 HP와 방어력. |
| 궁수 계열 | 궁수 (Archer) | 레인저 / 암살자 | 원거리 딜러. 빠른 공격속도. |
| 마법사 계열 | 마법사 (Mage) | 화염술사 / 빙결술사 | 강력한 광역 마법. 낮은 HP. |
| 사제 계열 | 사제 (Priest) | 치유사 / 신성기사 | 힐 및 버프. 파티의 핵심. |
| 연금술사 계열 | 연금술사 (Alchemist) | 폭탄사 / 독술사 | 상태이상 특화. 독특한 플레이스타일. |

### 3.2 레벨 시스템

- 레벨 범위: 1 ~ 100
- 경험치는 몬스터 처치 및 퀘스트 완료로 획득
- 레벨 구간별 스킬 포인트 지급
  - 1~20: 일반 구간 (솔로 가능)
  - 21~50: 중급 구간 (파티 권장)
  - 51~80: 상급 구간 (파티 필수)
  - 81~100: 던전 특화 구간

### 3.3 스탯

| 스탯 | 설명 |
|------|------|
| STR (힘) | 물리 공격력 증가 |
| AGI (민첩) | 공격속도, 회피율 증가 |
| INT (지력) | 마법 공격력 증가 |
| VIT (체력) | 최대 HP 증가 |
| WIS (지혜) | 최대 MP 및 MP 회복 속도 증가 |
| RES (저항) | 흡혈 저항 (적의 흡혈 효과 감소) |

> **RES(저항)** 스탯은 이 세계의 고유 스탯으로, 적들의 흡혈 효과를 줄이는 핵심 방어 지표다.

### 3.4 캐릭터 이미지 (초기)

- 직업별 고정 스프라이트 이미지 (픽셀아트 or 간단한 2D 일러스트)
- 장착 장비에 따라 외형 레이어가 변경되는 구조 (무기, 갑옷 최소 표현)
- 추후 고품질 캐릭터 이미지로 교체 가능한 모듈형 구조

## 4. 장비 시스템

### 4.1 장비 슬롯

- 무기 (Weapon)
- 머리 (Helmet)
- 상체 (Armor)
- 하체 (Pants)
- 장갑 (Gloves)
- 신발 (Boots)
- 반지 x2 (Ring)
- 목걸이 (Necklace)

### 4.2 장비 등급

| 등급 | 색상 | 특징 |
|------|------|------|
| 일반 | 회색 | 기본 스탯 |
| 고급 | 녹색 | 소폭 보너스 스탯 |
| 희귀 | 파랑 | 랜덤 추가 옵션 1~2개 |
| 영웅 | 보라 | 랜덤 추가 옵션 2~3개 |
| 전설 | 주황 | 고정 + 랜덤 옵션, 고유 패시브 |
| 심연 | 붉은 검정 | 던전 전용 드롭. 흡혈 관련 특수 능력 |

### 4.3 장비 강화

- 강화석 아이템으로 +1~+15까지 강화 가능
- +10 이상은 실패 시 등급 하락 위험
- 던전 전용 강화 재료 별도 존재

## 5. 몬스터 시스템

### 5.1 흡혈 메커니즘

모든 몬스터는 기본적으로 **흡혈(Blood Drain)** 능력을 보유한다.

| 흡혈 유형 | 설명 |
|-----------|------|
| 일반 흡혈 | 공격 시 가한 피해의 일부를 HP로 회복 |
| 광역 흡혈 | 주변 플레이어 전원에게서 지속적으로 HP 흡수 |
| 스킬 흡혈 | 특정 스킬 사용 시 대량 흡혈 |
| 마나 흡혈 | HP 대신 MP를 흡수 |
| 저주 흡혈 | 플레이어에게 흡혈 저주 디버프 부여 (지속 HP 감소) |

### 5.2 몬스터 목록 (예시)

| 이름 | 지역 | 특징 |
|------|------|------|
| 흡혈 슬라임 | 서광 평원 | 가장 약한 몬스터. 분열 가능. 일반 흡혈. |
| 피박쥐 | 서광 평원 | 빠른 이동. 군집 공격. 저주 흡혈. |
| 흡혈 늑대 | 안개 숲 | 패거리 사냥. 강력한 일반 흡혈. |
| 뱀파이어 나무 | 안개 숲 | 고정형. 광역 흡혈. 광범위 루트. |
| 사막 흡혈 벌레 | 붉은 사막 | 땅 속 잠복. 마나 흡혈 특화. |
| 혈족 전사 | 심연의 탑 | 인간형 몬스터. 스킬 흡혈 + 장비 보유. |
| 흡혈 골렘 | 심연의 탑 | 고체력 탱커형. 피해량의 50% 흡혈. |
| 심연 드래곤 | 균열의 핵 | 레이드 보스. 전체 흡혈 오라 + 즉사기. |

### 5.3 몬스터 등급

- 일반 / 정예(Elite) / 챔피언 / 보스 / 레이드 보스

## 6. 멀티플레이 시스템

### 6.1 솔로 플레이

- 필드 몬스터, 일반 퀘스트, 솔로 던전(1인용 소형 던전) 플레이 가능
- 다른 플레이어와 같은 필드에 존재하지만 직접 상호작용 선택 가능
- 상점, 경매장, 채팅 등 사교 시스템 이용 가능

### 6.2 파티 플레이 (4~5인)

- 파티 리더가 던전 입장 시작
- 파티원 모두 같은 인스턴스 던전 진입
- 보스 처치 시 파티 전원에게 드롭 아이템 분배
- 파티 버프 시스템: 사제 직업 버프가 파티 전원에게 적용
- 파티 채팅 / 근처 채팅 구분

### 6.3 던전 구조

- 던전은 층(Floor) 단위로 구성 (예: 심연의 탑 1~10층)
- 각 층에 일반 몬스터 구역 + 중간보스 + 최종보스
- 클리어 타임 기록 (주간 랭킹)
- 던전별 권장 레벨 및 파티 구성 제시

## 9. UI/UX 구성

- 메인 HUD: HP바, MP바, 스킬 슬롯(1~6), 미니맵
- 인벤토리: 그리드 방식 아이템 관리
- 캐릭터 창: 장비 슬롯 + 스탯 요약
- 파티 창: 파티원 HP/MP 실시간 표시 (사제 플레이 필수 요소)
- 던전 HUD: 층 정보, 몬스터 잔여 수, 타이머

## 10. 기술 스택 (제안)

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Phaser.js (2D 게임 프레임워크) or PixiJS |
| 실시간 통신 | Socket.io (멀티플레이 동기화) |
| 백엔드 | Node.js + Express |
| 데이터베이스 | PostgreSQL (캐릭터, 장비) + Redis (세션, 실시간) |
| 배포 | Docker + AWS or Vercel + Supabase |
