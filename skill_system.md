# 직업별 스킬 시스템 기획서

---

## 1. 스킬 시스템 개요

### 1.1 스킬 포인트

- 레벨업 시 **스킬 포인트(SP) 1점** 획득
- 전직 시 보너스 SP 지급 (1차 전직: +5, 2차 전직: +10)
- SP로 스킬 트리에서 스킬 해금 또는 레벨업

### 1.2 스킬 레벨

- 각 스킬은 최대 레벨 5
- 스킬 레벨업 시 위력/효과 증가, SP 1점 소비
- 스킬 초기화: 초기화 아이템(던전 보상)으로 전체 SP 환불 가능

### 1.3 스킬 유형

| 유형 | 설명 |
|------|------|
| **액티브** | 직접 발동. MP 소모. 쿨타임 있음. |
| **패시브** | 항상 적용. 스탯/확률 보정. |
| **버프** | 자신 또는 파티에게 효과 적용. |
| **궁극기** | 강력하지만 긴 쿨타임 (60~120초). |

### 1.4 스킬 슬롯

- 액티브 스킬 슬롯: 6개 (Q, W, E, R, T, Y)
- R은 궁극기 전용 슬롯
- 패시브는 슬롯 불필요, 자동 적용

## 7. 스킬 트리 구조

```
[1차 직업 스킬 6종]
        │
  레벨 25 달성 + 전직 아이템
        │
   ┌────┴────┐
[2차 직업A]  [2차 직업B]
 스킬 5종     스킬 5종
 궁극기 1종   궁극기 1종
```

- 2차 전직 후에도 1차 스킬 사용 가능 (슬롯 배치 자유)
- 2차 직업 스킬은 1차 스킬과 콤보 연계 가능하도록 설계

### 스킬 포인트 배분 예시 (레벨 50 기준)

| 구분 | SP 지출 |
|------|---------|
| 1차 직업 스킬 (6개 × 최대5) | 최대 30SP |
| 2차 직업 스킬 (5개 × 최대5) | 최대 25SP |
| 궁극기 (1개 × 최대5) | 최대 5SP |
| **레벨 50 보유 SP** | 50 + 전직 보너스 15 = **65SP** |

> 65SP로 모든 스킬을 최대로 찍는 것은 불가능. 플레이어가 빌드를 선택해야 함.

## 8. 파티 스킬 연계 (Combo)

| 연계 | 방법 | 효과 |
|------|------|------|
| 빙결 + 파쇄 강타 | 빙결술사가 빙결 → 전사가 강타 | 피해 +100% + 빙결 즉시 해제 |
| 독 + 폭발 | 독술사 중독 → 폭탄사 폭발 | 연쇄 폭발 발동, 피해 +50% 추가 |
| 속박 + 집중 사격 | 기사 방패 돌진 경직 → 레인저 집중 사격 | 이동 제한 상태에서 사격 배율 +30% |
| 흡혈 봉인 + 광역 딜 | 사제 흡혈 봉인 → 화염술사 업화 | 몬스터 회복 없이 전체 피해 적용 |
| 취약화 독 + 참격 | 독술사 취약화 → 버서커 무쌍 난무 | 방어력 감소 상태에서 전타 적용 |
